Privacy Policy

Last updated: March 2026

BookLedger is a mobile application designed to help users manage their personal book collections.

Information Collection

The app does not collect or store personal information on external servers. All data entered by users is stored locally on the user's device.

External Services

The app may retrieve book information (such as title, author, and description) from external services, including Google Books API. These requests are made only when the user performs a search or scans a book.

Data Storage

User data is stored locally on the device. Users may export and import their data manually using backup files.

Data Sharing

The app does not share user data with third parties.

Advertising

This app does not display advertisements.

Security

The app does not transmit sensitive personal data. Users are responsible for managing their backup files.

Changes to This Policy

This privacy policy may be updated in the future. Any changes will be posted on this page.

Contact

If you have any questions, please contact the developer.
